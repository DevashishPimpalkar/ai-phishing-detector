# AI-Powered Phishing Email Detector

This project is a simple AI-based tool that analyzes email content
and detects whether it is potentially phishing or safe.

## Why this project?
Phishing is one of the most common real-world cyber attacks.
This project focuses on practical detection using Python and ML.

## Tech Stack
- Python
- Scikit-learn
- TF-IDF
- Logistic Regression
